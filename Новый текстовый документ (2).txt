<p><b>H</b>, <b>h</b> ([<span class="IPA"><a href="/wiki/%D0%93%D0%BB%D1%83%D1%85%D0%BE%D0%B9_%D0%B3%D0%BB%D0%BE%D1%82%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%89%D0%B5%D0%BB%D0%B5%D0%B2%D0%BE%D0%B9_%D1%81%D0%BE%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B9" title="Глухой глоттальный щелевой согласный">h</a><a href="/wiki/%D0%9D%D0%B5%D0%BE%D0%B3%D1%83%D0%B1%D0%BB%D1%91%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D0%B4%D0%BD%D0%B5%D0%B3%D0%BE_%D1%80%D1%8F%D0%B4%D0%B0_%D0%BD%D0%B8%D0%B6%D0%BD%D0%B5%D0%B3%D0%BE_%D0%BF%D0%BE%D0%B4%D1%8A%D1%91%D0%BC%D0%B0" title="Неогублённый гласный переднего ряда нижнего подъёма">a</a><a href="/wiki/%D0%97%D0%BD%D0%B0%D0%BA_%D0%B4%D0%BE%D0%BB%D0%B3%D0%BE%D1%82%D1%8B_(%D1%81%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%9C%D0%A4%D0%90)" title="Знак долготы (символ МФА)">ː</a></span>])&nbsp;— восьмая буква базового <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" class="mw-redirect" title="Латинский алфавит">латинского алфавита</a>. В России эта латинская буква традиционно, особенно в науке, называется «аш» (по историческим причинам, названия большинства латинских букв были заимствованы из французского), хотя её исконное название в латинском языке произносится приблизительно «ха». В <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Английский алфавит">английском</a> языке буква называется «эйч», во <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Французский алфавит">французском</a>&nbsp;— «аш», в <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Немецкий алфавит">немецком</a>&nbsp;— «ха», в <a href="/wiki/%D0%9F%D0%BE%D1%80%D1%82%D1%83%D0%B3%D0%B0%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Португальский алфавит">португальском</a>&nbsp;— «ага́», в <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Итальянский алфавит">итальянском</a> — «а́кка», в <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" class="mw-redirect" title="Испанский алфавит">испанском</a> — «а́че».

<h2><span id=".D0.9F.D1.80.D0.BE.D0.B8.D0.B7.D0.BD.D0.BE.D1.88.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Произношение">Произношение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=H_(%D0%BB%D0%B0%D1%82%D0%B8%D0%BD%D0%B8%D1%86%D0%B0)&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Редактировать раздел «Произношение»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=H_(%D0%BB%D0%B0%D1%82%D0%B8%D0%BD%D0%B8%D1%86%D0%B0)&amp;action=edit&amp;section=2" title="Редактировать раздел «Произношение»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В большинстве языков с латинской письменностью обозначает звук [<span class="IPA"><a href="/wiki/%D0%93%D0%BB%D1%83%D1%85%D0%BE%D0%B9_%D0%B3%D0%BB%D0%BE%D1%82%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%89%D0%B5%D0%BB%D0%B5%D0%B2%D0%BE%D0%B9_%D1%81%D0%BE%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B9" title="Глухой глоттальный щелевой согласный">h</a></span>] (придыхание, простой выдох, <a href="/wiki/%D0%93%D0%BB%D1%83%D1%85%D0%BE%D0%B9_%D0%B3%D0%BB%D0%BE%D1%82%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%89%D0%B5%D0%BB%D0%B5%D0%B2%D0%BE%D0%B9_%D1%81%D0%BE%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B9" title="Глухой глоттальный щелевой согласный">глухой глоттальный щелевой согласный</a>).
</p><p>В <a href="/wiki/%D0%A0%D0%BE%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B5_%D1%8F%D0%B7%D1%8B%D0%BA%D0%B8" title="Романские языки">романских языках</a> (за исключением <a href="/wiki/%D0%91%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D0%BE-%D1%80%D0%BE%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B5_%D1%8F%D0%B7%D1%8B%D0%BA%D0%B8" title="Балкано-романские языки">балкано-романских</a>, <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%BB%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Валлонский язык">валлонского</a> и <a href="/wiki/%D0%93%D0%B0%D1%81%D0%BA%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Гасконский язык">гасконского</a>) не произносится. Во <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">французском языке</a> различаются «h немое» (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&nbsp;<span lang="fr" style="font-style:italic;">h muet</span>) и «h придыхательное» (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&nbsp;<span lang="fr" style="font-style:italic;">h aspiré</span>): они оба не произносятся, однако первое допускает <a href="/wiki/%D0%AD%D0%BB%D0%B8%D0%B7%D0%B8%D1%8F_(%D0%BB%D0%B8%D0%BD%D0%B3%D0%B2%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0)" title="Элизия (лингвистика)">элизию</a> и <a href="/wiki/%D0%9B%D1%8C%D0%B5%D0%B7%D0%BE%D0%BD_(%D0%BB%D0%B8%D0%BD%D0%B3%D0%B2%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0)" title="Льезон (лингвистика)">льезон</a>, а второе их не допускает.
</p><p>Также иногда не произносится в <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">английском</a> и <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">немецком</a> языках. В английском языке непроизносимое h встречается: в начале некоторых слов старофранцузского происхождения (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&nbsp;<span lang="en" style="font-style:italic;">heir, hour, honour, honest</span>); между ударным и безударным гласными (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&nbsp;<span lang="en" style="font-style:italic;">vehicle, annihilate</span>); в конце слов и перед согласными (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&nbsp;<span lang="en" style="font-style:italic;">ah, oh, cheetah, pharaoh, ohm, dahlia</span>). В немецком языке h не произносится (но указывает на долготу предшествующего гласного) в конце слов, перед согласными и перед безударным e (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a>&nbsp;<span lang="de" style="font-style:italic;">nah, Reh, Sohn, Kohl, gehen, mähen</span>).
</p><p>В <a href="/wiki/%D0%A7%D0%B5%D1%88%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Чешский язык">чешском</a>, <a href="/wiki/%D0%A1%D0%BB%D0%BE%D0%B2%D0%B0%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Словацкий язык">словацком</a> и <a href="/wiki/%D0%92%D0%B5%D1%80%D1%85%D0%BD%D0%B5%D0%BB%D1%83%D0%B6%D0%B8%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Верхнелужицкий язык">верхнелужицком</a> языках обозначает звук [<span class="IPA"><a href="/wiki/%D0%97%D0%B2%D0%BE%D0%BD%D0%BA%D0%B8%D0%B9_%D0%B3%D0%BB%D0%BE%D1%82%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%89%D0%B5%D0%BB%D0%B5%D0%B2%D0%BE%D0%B9_%D1%81%D0%BE%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B9" title="Звонкий глоттальный щелевой согласный">ɦ</a></span>] (<a href="/wiki/%D0%97%D0%B2%D0%BE%D0%BD%D0%BA%D0%B8%D0%B9_%D0%B3%D0%BB%D0%BE%D1%82%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%89%D0%B5%D0%BB%D0%B5%D0%B2%D0%BE%D0%B9_%D1%81%D0%BE%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B9" title="Звонкий глоттальный щелевой согласный">звонкий глоттальный щелевой согласный</a>, как украинское Г).
</p><p>В <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Польский язык">польском</a>, <a href="/wiki/%D0%A5%D0%BE%D1%80%D0%B2%D0%B0%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Хорватский язык">хорватском</a>, <a href="/wiki/%D0%A1%D0%BB%D0%BE%D0%B2%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Словенский язык">словенском</a> и <a href="/wiki/%D0%9B%D0%B0%D1%82%D1%8B%D1%88%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латышский язык">латышском</a> языках, а также в <a href="/wiki/%D0%9F%D0%B8%D0%BD%D1%8C%D0%B8%D0%BD%D1%8C" title="Пиньинь">пиньине</a> (системе передачи латиницей звуков <a href="/wiki/%D0%9F%D1%83%D1%82%D1%83%D0%BD%D1%85%D1%83%D0%B0" title="Путунхуа">стандартного китайского языка</a>) обозначает звук [<span class="IPA"><a href="/wiki/%D0%93%D0%BB%D1%83%D1%85%D0%BE%D0%B9_%D0%B2%D0%B5%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D0%B9_%D1%81%D0%BF%D0%B8%D1%80%D0%B0%D0%BD%D1%82" title="Глухой велярный спирант">x</a></span>] (<a href="/wiki/%D0%93%D0%BB%D1%83%D1%85%D0%BE%D0%B9_%D0%B2%D0%B5%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D0%B9_%D1%81%D0%BF%D0%B8%D1%80%D0%B0%D0%BD%D1%82" title="Глухой велярный спирант">глухой велярный спирант</a>, как русское Х).
</p><p>Также буква h часто используется в составе <a href="/wiki/%D0%94%D0%B8%D0%B3%D1%80%D0%B0%D1%84" title="Диграф">диграфов</a>.
</p>

В международных обозначениях физических единиц буква h является <a href="/wiki/%D0%9F%D1%80%D0%B8%D1%81%D1%82%D0%B0%D0%B2%D0%BA%D0%B8_%D0%A1%D0%98" title="Приставки СИ">десятичной приставкой</a> «<a href="/wiki/%D0%93%D0%B5%D0%BA%D1%82%D0%BE-" title="Гекто-">гекто</a>», обозначающей умножение единицы измерения на 100 (её кириллическое обозначение — «г»), а также обозначением <a href="/wiki/%D0%A7%D0%B0%D1%81" title="Час">часа</a> (его кириллическое обозначение — «ч»);</li>
<li>буквой <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r117753614"><span class="ts-math"><i>H</i></span> — <a href="/wiki/%D0%91%D0%BE%D0%B7%D0%BE%D0%BD_%D0%A5%D0%B8%D0%B3%D0%B3%D1%81%D0%B0" title="Бозон Хиггса">бозон Хиггса</a>, <a href="/wiki/%D0%93%D0%B0%D0%BC%D0%B8%D0%BB%D1%8C%D1%82%D0%BE%D0%BD%D0%B8%D0%B0%D0%BD_(%D0%BA%D0%B2%D0%B0%D0%BD%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B5%D1%85%D0%B0%D0%BD%D0%B8%D0%BA%D0%B0)" title="Гамильтониан (квантовая механика)">гамильтониан</a> и <a href="/wiki/%D0%A4%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D1%8F_%D0%93%D0%B0%D0%BC%D0%B8%D0%BB%D1%8C%D1%82%D0%BE%D0%BD%D0%B0" title="Функция Гамильтона">функцию Гамильтона</a>, <a href="/wiki/%D0%9D%D0%B0%D0%BF%D1%80%D1%8F%D0%B6%D1%91%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D1%8C_%D0%BC%D0%B0%D0%B3%D0%BD%D0%B8%D1%82%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BF%D0%BE%D0%BB%D1%8F" title="Напряжённость магнитного поля">напряжённость магнитного поля</a>, <a href="/wiki/%D0%AD%D0%BA%D0%B2%D0%B8%D0%B2%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%BD%D0%B0%D1%8F_%D0%B4%D0%BE%D0%B7%D0%B0" title="Эквивалентная доза">эквивалентную дозу</a>, <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D0%BE%D0%B7%D0%B8%D1%86%D0%B8%D1%8F_(%D1%84%D0%BE%D1%82%D0%BE)" title="Экспозиция (фото)">экспозицию</a>, <a href="/wiki/%D0%AD%D0%BD%D1%82%D0%B0%D0%BB%D1%8C%D0%BF%D0%B8%D1%8F" title="Энтальпия">энтальпию</a>. В международных обозначениях физических единиц буква H означает <a href="/wiki/%D0%93%D0%B5%D0%BD%D1%80%D0%B8_(%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0_%D0%B8%D0%B7%D0%BC%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%8F)" title="Генри (единица измерения)">генри</a> — единицу индуктивности (её кириллическое обозначение — «Гн»).</li></ul></li>
<li>В <a href="/wiki/%D0%93%D0%B5%D0%BE%D0%B4%D0%B5%D0%B7%D0%B8%D1%8F" title="Геодезия">геодезии</a> буквой <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r117753614"><span class="ts-math"><i>h</i></span> обозначают <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%B2%D1%8B%D1%88%D0%B5%D0%BD%D0%B8%D0%B5" class="mw-redirect" title="Превышение">превышение</a>.</li>
<li>В <a href="/wiki/%D0%9C%D1%83%D0%B7%D1%8B%D0%BA%D0%B0" title="Музыка">музыке</a> (в русской и немецкой системе <a href="/wiki/%D0%91%D1%83%D0%BA%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BD%D0%BE%D1%82%D0%B0%D1%86%D0%B8%D1%8F" title="Буквенная нотация">буквенной нотации</a>) H и h обозначают <a href="/wiki/%D0%9D%D0%BE%D1%82%D0%B0_(%D0%BC%D1%83%D0%B7%D1%8B%D0%BA%D0%B0)" title="Нота (музыка)">ноту</a> «<a href="/wiki/%D0%A1%D0%B8_(%D0%BD%D0%BE%D1%82%D0%B0)" title="Си (нота)">си</a>». В обозначениях <a href="/wiki/%D0%90%D0%BA%D0%BA%D0%BE%D1%80%D0%B4" title="Аккорд">аккордов</a> H без индексов — си-мажорный аккорд, а h иногда обозначает си-минорный аккорд (чаще это трезвучие обозначается H<sub>m</sub>)</li>
<li>В <a href="/wiki/%D0%A5%D0%B8%D0%BC%D0%B8%D1%8F" title="Химия">химии</a> H является символом элемента <a href="/wiki/%D0%92%D0%BE%D0%B4%D0%BE%D1%80%D0%BE%D0%B4" title="Водород">водорода</a>.</li>
<li>В <a href="/wiki/%D0%A4%D0%B0%D1%80%D0%BC%D0%B0%D1%86%D0%B5%D0%B2%D1%82%D0%B8%D0%BA%D0%B0" title="Фармацевтика">фармацевтике</a> витамин H — <a href="/wiki/%D0%91%D0%B8%D0%BE%D1%82%D0%B8%D0%BD" title="Биотин">биотин</a>.</li></ul>
